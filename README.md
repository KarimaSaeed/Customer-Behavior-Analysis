# Customer-Behavior-Analysis
This project focuses on applying exploratory data analysis (EDA) and data mining techniques to uncover patterns, relationships, and insights within a structured dataset.
The goal is to analyze the data, identify trends, and extract meaningful information to support data-driven decision-making.

---

## 🔍 Exploratory Data Analysis
- Performed univariate and multivariate analysis
- Explored feature distributions and relationships
- Identified patterns and correlations in the dataset
- Visualized insights using Matplotlib and Seaborn

---

## 🧹 Data Preprocessing
- Handled missing values and inconsistencies
- Cleaned and prepared the dataset for analysis
- Transformed data for better interpretation

---

## 📈 Key Insights
- Discovered relationships between variables  
- Identified trends and patterns within the data  
- Extracted insights that support data-driven decisions  

---

## 🛠️ Technologies Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
